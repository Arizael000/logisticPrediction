📦 Resumen Ejecutivo: Predicción Logística para Ventas
Este proyecto implementa un modelo de Machine Learning (Regresión o Series Temporales) para predecir las demandas logísticas futuras basándose en datos históricos de ventas y factores externos (estacionalidad, promociones, etc.).

🎯 Objetivo Principal
Predecir con precisión moderada la demanda de inventario en periodos futuros (ej. semana/mes siguiente), permitiendo una planificación proactiva de los recursos logísticos (almacenes).

🛠️ Metodología
Datos: Utilización de datos de ventas históricos y variables de tiempo (estacionalidad, día festivo).

Modelado: Se empleó Regresión logística para manejar la naturaleza de las series temporales de la demanda.

Métricas: El rendimiento se midió con Error Absoluto Medio (MAE) para cuantificar el error de predicción en términos de unidades o volumen logístico.

Impacto: La implementación de la predicción aumentó las ventas próximas y minimizó los riesgos de falta de stock o retrasos debido a capacidad insuficiente.
